# FDAL-SSA: Fitness-Distance Adaptive Levy SSA 改进原理

## 一、背景：为什么之前所有Levy变体都失败了

在FDAL-SSA之前，对SSA做过以下改进尝试：

| 变体 | 策略 | F1(单峰) | F2(单峰) | F3(多峰) | F4(多峰) |
|------|------|----------|----------|----------|----------|
| ISSA_Levy | Tent+Levy+Cauchy三策略融合 | -16400% | -595% | -1900% | +80% |
| ISSA_QuanZhong | Tent+自适应权重+Cauchy | 持平 | 持平 | +3% | +50% |
| Levy_Danger | Levy只在发现者危险态 | -11% | -4% | +2.5% | +52% |
| SmartSSA | 多策略侵入式改动 | -154% | -389% | -10355% | +47% |
| Conservative_v2 | Tent+反射边界+微Cauchy | +9% | -4% | +6% | +77% |
| PaperStyle | Tent+Levy扰动+Cauchy后处理 | +4% | -6% | +12% | +76% |
| **FDAL-SSA** | **距离自适应有向Levy** | **+100%** | **+60%** | **+84%** | **+79%** |

### 核心规律

1. **单峰函数(F1/F2)极其敏感** — 原版SSA发现者的安全态公式 `X × exp(-r/(t+1))` 指数衰减向0收敛已接近最优，任何扰动都是破坏
2. **多峰函数(F4 Rastrigin)极其宽容** — 几乎任何探索增强都有效，因为原版SSA极易陷入局部最优
3. **Levy飞行"有用但不该用的地方用"是失败的根源** — 对各向同性随机步不加选择地应用，破坏了SSA的核心收敛机制

### Levy失败的三条根因

**根因1: 各向同性随机（最致命）**

Mantegna算法生成的Levy步长在30维空间中方向完全随机。在D=30的高维空间里，随机向量与梯度方向近乎正交的概率极高。结果：99%的Levy跳跃朝错误方向，只在极偶然时有效。

**根因2: 无差别扰动**

对所有个体（包括精英）施加相同扰动策略。在Sphere函数上，精英已接近全局最优(0向量)，此时一个不巧的大跳跃直接摧毁高精度解。

**根因3: 步长一刀切**

`scale = 0.01 × (1-t/T) × |ub-lb|` 这种公式对所有个体一样，不管它离最优有多远。当种群已收敛到10^-4精度时，0.01的步长相当于原子弹炸蚂蚁。

## 二、FDAL-SSA的设计哲学

**核心原则**: 识别并保护原版SSA中最有价值的机制，只在低风险、低影响的环节精准植入改进。

### 改动边界（四象限原则）

```
               改动量
                  ↑
    大改(不碰)   │   小改(可以碰)
   ──────────────┼──────────────────→ 风险
    大改(不碰)   │   小改(可以碰)
```

- **安全态公式(80%更新)** → 改动风险极高 → **一字不改**
- **跟随者公式** → 改动风险高 → **一字不改**
- **警戒者公式** → 改动风险中等 → **一字不改**
- **危险态公式(20%更新)** → 风险低，原版就是随机散步 → **精准替换**
- **初始化** → 风险极低，消融验证无害 → Tent
- **边界处理** → 风险极低 → 反射边界
- **后处理** → 风险极低，有更好没更不差 → 极微柯西

## 三、五大策略详解

### 策略1: Tent混沌初始化

**原版**: `X = uniform(lb, ub)` — 可能随机扎堆或留空洞

**改为**: Tent混沌映射生成初始种群

```
x_{n+1} = { x_n / 0.5,            x_n < 0.5
          { (1-x_n) / (1-0.5),    x_n ≥ 0.5
```

**原理**: Tent映射对初值极端敏感（混沌特性），产生的序列在[0,1]间遍历性好、分布均匀。对每个维度独立生成一列Tent序列，映射到搜索空间后，40个个体在30维空间中分布更均匀，不浪费初始评估。

**消融证据**: 单用Tent初始化，F1 Mean改善+9%，F4改善+13%，其他函数不劣化。

---

### 策略2: 发现者危险态 — 距离自适应有向Levy（最核心）

这是FDAL-SSA最关键的创新，包含四层机制：

#### 第1层: 锁定改动范围

SSA发现者有两种状态:

| 状态 | 条件(ST=0.8) | 原版公式 | 作用 | 占比 |
|------|-------------|----------|------|------|
| 安全态 | r2 < 0.8 | `X × exp(-r/(t+1))` | 向0指数衰减 | ~80% |
| 危险态 | r2 ≥ 0.8 | `X + N(0,1)` | 纯随机散步 | ~20% |

**安全态一字不改**（第222行），因为 `X × exp(-r/(t+1))` 是SSA单峰优越性的根本。之前所有失败变体都在这个公式上动了手脚。

**只在危险态做替换**，因为原版危险态就是无方向的高斯散步，换了损失可控。

#### 第2层: 距离自适应步长（解决"原子弹炸蚂蚁"）

```python
norm_dist = mean( |X_i - X_best| / search_range )   # 归一化距离
distance_factor = sqrt( clip(norm_dist, 0.001, 1.0) )
scale = base_scale × distance_factor × ...
```

**核心公式: 步长 ∝ sqrt(个体到最优的距离)**

| 场景 | 距离 | distance_factor | 效果 |
|------|------|-----------------|------|
| 远处个体 | norm_dist=10% | ≈0.32 | 基步长×0.32，充分探索 |
| 近处个体 | norm_dist=0.01% | ≈0.01 | 基步长×0.01，几乎不动 |
| 单峰收敛后 | norm_dist→0 | →0.032(clip) | 保护高精度收敛 |
| 多峰探索中 | norm_dist保持 | 保持 | 持续有意义的探索 |

**这是F1/F2从之前"Levy必降"变成"Levy还能升"的根本原因**。在Sphere函数收敛后期，所有个体到0的距离极小 → 步长几乎为零 → Levy退化 → 安全态独占 → 精度不打折。

#### 第3层: 有向Levy（解决"99%朝错误方向"）

**原版各向同性Levy**:
```
X_new = X + scale × Levy()    # 在D=30下随机方向, 99%无用
```

**FDAL有向Levy**:
```python
direction = (X_best - X_i) / |X_best - X_i|    # 指向最优的单位向量
mix_ratio = 0.3 + 0.4 × (t/T)                   # 0.3 → 0.7

step = scale × (mix_ratio × direction + (1-mix_ratio) × levy_norm)
```

- 前期t=0: mix=0.3 → 30%有向 + 70%随机 → 偏探索
- 中期t=250: mix=0.5 → 50%有向 + 50%随机 → 平衡
- 后期t=500: mix=0.7 → 70%有向 + 30%随机 → 偏收敛

**为什么有向+随机混合比纯有向好**: 纯梯度方向容易陷入局部最优。叠加Levy的肥尾跳跃，偶尔的极端大步长能跳出浅层局部最优。混合比例线性增长，让算法从"多探索少利用"自然过渡到"多利用少探索"。

#### 第4层: 成功历史自适应 + 多样性衰减（闭环控制）

```python
# 成功历史: 追踪Levy扰动后实际改善的比例
success_ratio = levy_successes / levy_attempts
if success_ratio > 0.25:   boost *= 1.05    # 25%+成功率 → 加大
if success_ratio < 0.10:   boost *= 0.95    # <10%成功率 → 减小

# 多样性衰减: 种群抱团时自动缩步长
diversity = mean(std(X, axis=0))
damping = clip(diversity / 1.0, 0.1, 1.0)

# 最终步长 = 基步长 × 距离因子 × 历史调节 × 多样性衰减
scale = base_scale × distance_factor × boost × damping
```

这是一个**双回路闭环控制器**:
- **回路1(成功历史)**: Levy成功率 → boost → 平衡探索利用
- **回路2(多样性)**: 种群分散度 → damping → 保护收敛

当种群在单峰上高度收敛时: diversity→0.01 → damping=0.1 → 步长再缩10倍 → 双重保险。

### 策略3: 镜像反射边界

**原版clip问题**: 超出边界的个体被截断到边界上，在边界面上"粘住"，产生边界聚集效应。

```python
# 原版
if X > ub: X = ub       # 粘在边界

# FDAL: 镜像反射
if X > ub: X = 2×ub - X  # 反弹回内部
if X < lb: X = 2×lb - X
```

**原理**: 维持个体分布的连续性，不浪费评估次数在边界上。对Sphere等最优在中心的函数尤其重要。

### 策略4: 全局最优极小幅柯西变异（后处理）

```python
if random < 0.25:                          # 25%概率触发
    cauchy_scale = 0.0005 × (1-t/T) × |ub-lb|
    X_mutant = X_best + cauchy_scale × Cauchy()
    if better: replace
```

**量级**: 对[-100,100]的F1，前期scale≈0.05，后期≈0.0005

**原理**: 柯西分布肥尾，0.0005的量级下99%变异<0.01（无害），但偶尔的中等跳跃能帮全局最优逃离浅层局部最优（有益）。

**消融证据**: `ablation3_Cauchy_vs_SSA.csv` — 单用柯西变异所有函数不劣化，F3微改善。

---

### 策略5: 停滞检测与强化探索（安全网）

```python
if 连续30代无改进:
    对最差50%个体:
        step = 0.03 × clip(|X_i - X_best|, ...) × (0.5×direction + 0.5×levy)
```

**触发条件**: 全局最优30代不动

**执行**: 只扰动排名后50%的个体（保护精英），用有向Levy中等步长扰动

**作用**: 安全网 — 正常情况不触发（不断有改进），只在算法卡死时激活。如果触发经常发生，说明Levy的boost参数需要调大。

## 四、成功要素总结

```
FDAL-SSA成功 = 精准诊断 + 最小改动 + 多层自适应

精准诊断:
  ├─ 安全态公式 → SSA核心优势 → 保护不动
  ├─ 危险态公式 → 原版就弱 → 精准替换
  ├─ 各向同性 → Levy失败主因 → 有向化
  └─ 无差别扰动 → 精度破坏 → 距离自适应

最小改动:
  ├─ 只改4处（初始化/危险态/边界/后处理）
  └─ 安全态/跟随者/警戒者 100%原版

多层自适应（4层保护网）:
  ├─ Layer1: 距离因子 — 收敛时步长自动归零
  ├─ Layer2: 多样性衰减 — 种群抱团时再降10倍
  ├─ Layer3: 成功历史 — 无效则自动缩,有效则自动加
  └─ Layer4: 停滞检测 — 兜底安全网
```

## 五、与之前变体的本质区别

所有之前变体的思路: "找到一个更好的通用公式来替代SSA的部分或全部更新规则"
FDAL-SSA的思路: "保留SSA的核心公式，在风险最低的20%更新空间中，用多层自适应控制器来调节Levy的强度和方向"

前者是**公式驱动**（用一个方案解决所有情况），后者是**控制驱动**（让算法根据自身状态决定要不要探索、怎么探索）。

这就是为什么FDAL-SSA是唯一在全部4个基准函数上同时超越原版SSA的变体。
